## Built-in Classes Enhancement

_Libraries for enhancing Python built-in classes._

<CardGroup cols={2}>
  <Card title="attrs" icon="layer-group" href="https://github.com/python-attrs/attrs">
    Replacement for `__init__`, `__eq__`, `__repr__`, etc. boilerplate in class definitions.
  </Card>
  <Card title="bidict" icon="arrow-right-arrow-left" href="https://github.com/jab/bidict">
    Efficient, Pythonic bidirectional map data structures and related functionality.
  </Card>
  <Card title="box" icon="box" href="https://github.com/cdgriffith/Box">
    Python dictionaries with advanced dot notation access.
  </Card>
</CardGroup>

```python
# attrs — clean class definitions
import attrs

@attrs.define
class Point:
    x: float
    y: float

    def distance_from_origin(self) -> float:
        return (self.x ** 2 + self.y ** 2) ** 0.5

p = Point(3.0, 4.0)
print(p.distance_from_origin())  # 5.0
```

```python
# toolz functional utilities
from toolz import curry, compose, pipe

@curry
def add(x, y):
    return x + y

add5 = add(5)
result = pipe([1, 2, 3, 4], lambda lst: map(add5, lst), list)
print(result)  # [6, 7, 8, 9]
```

```python
# asyncio — async programming
import asyncio

async def fetch_data(url: str) -> dict:
    await asyncio.sleep(0.1)  # simulate network call
    return {"url": url, "status": 200}

async def main():
    tasks = [fetch_data(f"https://api.example.com/{i}") for i in range(5)]
    results = await asyncio.gather(*tasks)
    print(f"Fetched {len(results)} items")

asyncio.run(main())
```

## Date and Time

_Libraries for working with dates and times._

<CardGroup cols={2}>
  <Card title="dateparser" icon="calendar-check" href="https://github.com/scrapinghub/dateparser">
    A Python parser for human-readable dates in dozens of languages.
  </Card>
  <Card title="dateutil" icon="calendar-days" href="https://github.com/dateutil/dateutil">
    Extensions to the standard Python datetime module.
  </Card>
  <Card title="pendulum" icon="clock" href="https://github.com/python-pendulum/pendulum">
    Python datetimes made easy.
  </Card>
  <Card title="zoneinfo" icon="earth-americas" href="https://docs.python.org/3/library/zoneinfo.html">
    (Python standard library) IANA time zone support. Brings the tz database into Python.
  </Card>
</CardGroup>

```python
# pendulum — better datetimes
import pendulum

now = pendulum.now("Europe/London")
past = pendulum.datetime(2024, 1, 1)

print(now.diff_for_humans(past))  # "1 year ago"
print(now.in_timezone("Asia/Tokyo").to_datetime_string())
```
