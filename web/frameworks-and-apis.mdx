Python has a rich ecosystem of web frameworks ranging from batteries-included full-stack options to minimal microframeworks and high-performance async alternatives. This page covers frameworks for building web applications and APIs, serving them in production, handling real-time connections, templating HTML, and managing static assets.

### Django example

```python
from django.db import models

class Article(models.Model):
    title = models.CharField(max_length=200)
    body = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)
```

### Flask example

```python
from flask import Flask

app = Flask(__name__)

@app.route("/")
def hello_world():
    return "<p>Hello, World!</p>"
```

### FastAPI example

```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/items/{item_id}")
async def read_item(item_id: int, q: str | None = None):
    return {"item_id": item_id, "q": q}
```
