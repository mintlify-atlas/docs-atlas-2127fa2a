  </Card>
  <Card title="black" icon="circle" href="https://github.com/psf/black">

  <Card title="isort" icon="arrow-down-a-z" href="https://github.com/PyCQA/isort">
    A Python utility / library to sort imports.
  </Card>
</CardGroup>

```bash ruff_usage.sh
# ruff — fast linter and formatter
ruff check .          # lint
ruff check . --fix    # auto-fix
ruff format .         # format code
```

```python black_example.py
# black formatter (run via CLI)
# black my_file.py
# Before:
x = {'a': 1, 'b': 2}
y = [1,2,3]
# After black:
x = {"a": 1, "b": 2}
y = [1, 2, 3]
```
