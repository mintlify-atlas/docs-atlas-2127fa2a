### Basic pytest example

```python pytest_example.py
import pytest

def add(a, b):
    return a + b

def test_add():
    assert add(2, 3) == 5
    assert add(-1, 1) == 0
    assert add(0, 0) == 0

# Run with: pytest pytest_example.py -v
```

### Property-based testing with hypothesis

```python hypothesis_example.py
from hypothesis import given, strategies as st

@given(st.integers(), st.integers())
def test_add_is_commutative(a, b):
    assert a + b == b + a
```

### pytest with fixtures and mocking

```python mock_example.py
from unittest.mock import patch, MagicMock
import pytest

@pytest.fixture
def mock_api():
    with patch("myapp.api_client") as mock:
        mock.get.return_value = MagicMock(status_code=200, json=lambda: {"data": []})
        yield mock

def test_fetch_data(mock_api):
    from myapp import fetch_data
    result = fetch_data()
    assert result == []
    mock_api.get.assert_called_once()
```

### Object Factories

<CardGroup cols={2}>
  <Card title="factory_boy" icon="industry" href="https://github.com/FactoryBoy/factory_boy">
    A test fixtures replacement for Python.
  </Card>
  <Card title="polyfactory" icon="cubes" href="https://github.com/litestar-org/polyfactory">
    Mock data generation library with support to classes (continuation of `pydantic-factories`).
  </Card>
</CardGroup>

```python factory_boy_example.py
import factory
from myapp.models import User

class UserFactory(factory.Factory):
    class Meta:
        model = User
    name = factory.Faker("name")
    email = factory.Faker("email")
    age = factory.Faker("random_int", min=18, max=80)

user = UserFactory()
```
