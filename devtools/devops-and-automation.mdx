## Task Queues

<CardGroup cols={2}>
  <Card title="celery" icon="layer-group" href="https://github.com/celery/celery">
    An asynchronous task queue/job queue based on distributed message passing.
  </Card>
  <Card title="dramatiq" icon="bolt" href="https://github.com/Bogdanp/dramatiq">
    A fast and reliable background task processing library for Python 3.
  </Card>
  <Card title="rq" icon="list" href="https://github.com/rq/rq">
    Simple job queues for Python.
  </Card>
  <Card title="huey" icon="clock" href="https://github.com/coleifer/huey">
    Little multi-threaded task queue.
  </Card>
</CardGroup>

```python celery_example.py
from celery import Celery

app = Celery("tasks", broker="redis://localhost:6379/0")

@app.task
def add(x, y):
    return x + y

# Call asynchronously:
result = add.delay(4, 4)
print(result.get(timeout=10))  # 8
```

```python loguru_example.py
logger.add("app.log", rotation="10 MB", retention="7 days")
logger.info("Server started on port {port}", port=8080)
logger.error("Database connection failed: {err}", err="timeout")
```
